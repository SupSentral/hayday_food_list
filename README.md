# 🌾 Hay Day Food List

Website koleksi makanan dan produk Hay Day yang menampilkan daftar item lengkap berdasarkan level, mesin produksi, dan kategori. Dibuat untuk membantu pemain mencari informasi item dengan lebih cepat dibandingkan membuka wiki secara manual.

🔗 Demo: https://supsentral.github.io/hayday_food_list/

---

## ✨ Fitur

- 📋 Daftar lengkap makanan dan produk Hay Day
- 🔍 Pencarian instan berdasarkan nama item
- 🏭 Filter berdasarkan mesin produksi
- ⭐ Informasi level unlock setiap item
- 🍞 Tampilan emoji untuk identifikasi item lebih cepat
- 📱 Responsive untuk desktop dan mobile
- ⚡ Berjalan 100% di browser tanpa backend
